590. N叉树的后序遍历

难度简单119

给定一个 N 叉树，返回其节点值的后序遍历。

例如，给定一个 3叉树 :

返回其后序遍历: [5,6,3,2,4,1]

递归

LinkedList<Integer> list = new LinkedList<>();
public List<Integer> postorder(Node root) {

    dfs(root);
    return  list;
}

private void dfs(Node root) {
    if (root==null){
        return;
    }
    for (Node chlid : root.children) {
        dfs(chlid);
    }
    list.add(root.val);
}

迭代

  • 使用一个栈来得到后序遍历
  • 首先把跟节点放入栈中,当我们每次从栈顶取出一个节点u点,,就把就把 u 的所有子节点顺序推入栈中。
  • u 的子节点从左到右为 v1, v2, v3,,那么推入栈的顺序为 v1, v2, v3，
  • 就保证了下一个遍历到的节点（即 u 的第一个子节点 v3）出现在栈顶的位置。在遍历结束之后，我们把遍历结果反转，就可以得到后序遍历。
/**
 * N叉树的后序遍历
 */
public class LEE590 {
    public List<Integer> postorder11(Node root) {
        LinkedList<Node> stack = new LinkedList<>();
        LinkedList<Integer> output = new LinkedList<>();
        if (root == null) {
            return output;
        }
        stack.add(root);
        while (!stack.isEmpty()) {
            Node node = stack.pollLast();
            output.addFirst(node.val);
            for (Node item : node.children) {
                if (item != null) {
                    stack.add(item);
                }
            }
        }
        return output;
    }
