十大排序-希尔排序

2020-11-28 -算法 403 words
排序 十大排序

希尔排序（Shell Sort)

基本思想:

  • 希尔排序也是一种插入排序,它是简单插入排序经过改进之后的一个更高效的版本,**也称为缩小增量排序**,
  • 该算法是冲破O(n2）的第一批算法之一,他与插入排序的不同之处在于,他会优先比较距离较远的元素

基本思路:

  • 希尔排序是把记录按下标的一定增量分组,对每组使用直接插入排序算法排序:
  • 随着增量逐渐减少,每组包含的关键字越多,
  • 当增量减之到1时,整个文件恰被分成一组， 算法便终止 
public class ShellSortDemo {
    public static void main(String[] args) {
        int[] arr = {8, 9, 1, 7, 2, 3, 5, 4, 6, 0};
        System.out.println(Arrays.toString(shellSort(arr)));
    }
    //对交换式的希尔排序进行优化->移位法
    public static int[] shellSort(int[] nums) {
        int gap;/*
        增量gap, 并逐步的缩小增量
        每次缩小二倍
        */
        for (gap = nums.length / 2; gap > 0; gap /= 2) {
            for (int i = gap; i < nums.length; i++) {
                    int j = i;
                    int temp = nums[j];//比较两个数的 后面那一部分
                    //nums[j - gap]表示前面的数
                    while (j - gap >= 0 && temp < nums[j - gap]) {
                        //移动
                        //将前面小的 移动到后面
                        nums[j] = nums[j - gap];
                        j -= gap;
                    }
                    //当退出while后，就给temp找到插入的位置
                    nums[j] = temp;
                }
            }
        return nums;
    }
}

总结:

平均时间复杂度O(nlog^n^)

最好情况 O(nlog^2^n)

最坏情况 O(nlog^2^n)

空间复杂度 O(1)

排序方式 ln-place

稳定性 不稳定
