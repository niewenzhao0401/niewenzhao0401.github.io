正则表达式 基础 java

正则表达式

  • 正则表达式,即规则表达式
  • 模式匹配的语法,用特定的符号,表示某种规则,取匹配某个字符串
  • 如果字符串与规则相匹配,返回true,否则返回false
s.matches("\\d");//匹配

  • \\d 匹配数字

    [4-9] 控制区间
 [4567] 只能匹配出现数字的一次
 X?  一次或者一次也没有
 X*  零次或者多次
 X+  一次或者多次[即不能为空]

  • //表示次数
    X{n] 恰好n次
    X{n,} 至少n次
    x{n,m} 至少n次,最多m次

*\\d 匹配数字 X 零次或者多次**

String s1 = "13667890876";
boolean c1= s1.matches("\\d*");//匹配//X*  零次或者多次
System.out.println(c1);//true

*X? 一次或者一次也没有 X 零次或者多次**

String s2="1";
boolean c2=s2.matches("\\d?");//X?  一次或者一次也没有
System.out.println(c2);//true
String s3="111";
boolean c3=s3.matches("\\d+");// X*  零次或者多次
System.out.println(c3);//true

匹配11位电话号 只能1开头, 第二位只能是35789

boolean c2 = s1.matches("[1][35789]\\d{9}");
System.out.println(c2);//true

//[4-9] 控制区间
//[4567] 只能匹配出现数字的一次

String s4 = "9";
boolean c4 = s4.matches("[4-9]"); //左边 右边 闭合
System.out.println(c4);//true

五到九位的qq号

String s4="1443906329";
boolean c4=s4.matches("[1-9]\\d{5,9}");
System.out.println("QQQQqq");
System.out.println(c4);
  • \d 数字:[\1-9]
    \D 非数字:[^0-9]
    \s 空白字符
    \S 非空白字符
    \w 单词字符
    \W 非单词字符

*空白字符 \s 多个**

String d1 = "   ";
boolean e1 = d1.matches("\\s*");//小写字母  // 空白字符   \\s* 多个
System.out.println("空白字符");
System.out.println(e1);//true

非空白字符

String d2 = "fadsda";
boolean e2 = d2.matches("\\S*");//大写字母// 非空白字符
System.out.println("非空白字符");
System.out.println(e2);//true
System.out.println("----");

//\w 单词字符：[a-zA-Z_0-9]
//\W 非单词字符：[^\w]

String d3 = "werwer";
boolean e3 = d3.matches("\\w*");// 单词字符
System.out.println("单词字符");
System.out.println(e3);
System.out.println("----");

String d4 = "@#$@#$@#$@#%@#%";
boolean e4 = d4.matches("\\W*");// 非单词字符
System.out.println("非单词字符");
System.out.println(e4);
System.out.println("----");

通用邮箱

  • // . 表示任意字符
  • //需要时对其转义 \.
  • //(com|com.cn)表示com 或者 com.cn
String s5 = "123sas231@qq.com.cn";
//  .  表示任意字符
//需要时对其转义  \\.
//(com|com.cn)表示com 或者 com.cn
boolean c5 = s5.matches("\\w{6,15}@\\w{2,6}\\.(com|com.cn)");
System.out.println("youxiang");
System.out.println(c5);
String s6 = "asada23j424k231";
String[] c6 = s6.split("\\d");
System.out.println(Arrays.toString(c6));
System.out.println(s6.replaceFirst("\\d", "W"));//转换第一个
