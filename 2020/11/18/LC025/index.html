25. K 个一组翻转链表

难度困难793收藏分享切换为英文接收动态反馈

给你一个链表，每 k 个节点一组进行翻转，请你返回翻转后的链表。

k 是一个正整数，它的值小于或等于链表的长度。

如果节点总数不是 k 的整数倍，那么请将最后剩余的节点保持原有顺序。

示例：

给你这个链表：1->2->3->4->5

当 k = 2 时，应当返回: 2->1->4->3->5

当 k = 3 时，应当返回: 3->2->1->4->5

说明：

  • 你的算法只能使用常数的额外空间。
  • 你不能只是单纯的改变节点内部的值，而是需要实际进行节点交换

栈

有一道经典的题目，反转链表，将1->2->3->4变成4->3->2->1。反转链表的一个解法是用栈作为辅助结构，将[1,2,3,4]放入栈中，然后依次弹出来，再把弹出来的节点串联起来，这样就变成4->3->2->1了。

对于k个一组反转，我们也可以用同样的方式，只不过这次不是将全部节点放入栈中，而是将k个节点放入栈中，之后再弹出k个节点，并将他们依次串联起来，这样就达到了反转k个节点的效果了。

有个细节需要注意下，比如链表是:
1->2->3->4->5->6->7，k=3时，整个链表遍历完后，栈的个数不等于k，此时栈中只有一个元素7。判断栈长度不等于k直接退出就可以了。

当然用栈实现其实并不符合题意，只当扩展下思路。

public ListNode reverseKGroup(ListNode head, int k) {
    if (head == null || k <=0) {
        return head;
    }
    Stack<ListNode> stack = new Stack<>();
    //设置一个新链表的头
    ListNode dummy = new ListNode(-1);
    //新节点指向原头
    dummy.next = head;
    //对于新链表设置一个辅助节点
    ListNode p = dummy;
    int n = k;
    while (p != null && p.next != null) {
        //设置一个辅助节点来进行栈的处理
        ListNode temp = p.next;
        while (temp != null && n > 0) {
            //进行入栈
            stack.add(temp);
            temp = temp.next;
            --n;
        }
        //出站
        //标记出来第一个节点
        //12  就标记的是2
        //备份K+1 节点,用于后面的串的作用
        ListNode nextNode = stack.peek().next;
        //这里n等于零就表示正好存放k个节点
        if (n == 0) {
            while (stack.size() > 0) {
                p.next = stack.pop();
                p = p.next;
            }
        }else {
            break;
        }
        p.next=nextNode;
        n=k;

    }
    return dummy.next;
}

迭代

思路

  • 1->2->3->4->5->6->7
  • 反转K个,由于遍历k个链表,然后将第k个节点的next指向空(防止循环)
  • 这样就得到了长度为k的单链表了。1->2->3 4->5->6->7
  • 然后反转所截的
  • 如果遍历完整个链表，发现最后几个节点的长度不足k，那么直接退出就可以了。
  • 把长度k的链表前后指向切断，这样就不会出现循环指向了。
class Solution {
    public ListNode reverseList(ListNode head) {
        ListNode pre = null;//新节点
        ListNode cur = head;//辅助结点进行反转的
        while (cur != null) {
            ListNode temp = cur.next;////标记下一次需要反转的节点
            cur.next = pre;//已经开始反转
            //移动节点  准备下一次反转
            pre = cur;
            cur = temp;
        }
        return pre;
    }

    public ListNode reverseKGroup(ListNode head, int k) {
        if (head == null || k < 0) {
            return head;
        }

        ListNode dummy = new ListNode(-1);
        //新节点指向头节点
        dummy.next = head;
        //设置辅助指针进行遍历
        ListNode p = dummy;
        int n = k;
        while (p.next != null) {
            //辅助指针进行遍历
            ListNode temp = p;
            while (temp != null && temp.next != null && n > 0) {
                //在合法范围之内遍历到  k的位置上
                temp = temp.next;
                --n;
            }

            if (n == 0) {
                //报先记录下temp.next,方便后面链接链表
                ListNode nextNode = temp.next;
                //得到开始要遍历的节点
                //假设链表为1->2->3->4，下面的nextNode就是4，tail是1
                ListNode tail = p.next;
                //反转之前需要将下一个节点保存，并设置next为空，防止循环指向
                temp.next = null;
                //进行反转
                //p 也即是dummy指向反转后的节点 dummy->2->1
                p.next = reverseList(tail);
                //翻转后头节点变到最后。通过.next把断开的链表重新链接。
                //1<-2 3->4 转换成为    2->1->3->4
                tail.next = nextNode;
                //移动p节点进行下一个遍历
                p = tail;
                n = k;
            } else {
                break;
            }
        }
        return dummy.next;
    }
