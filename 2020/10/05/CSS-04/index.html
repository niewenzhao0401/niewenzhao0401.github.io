CSS–伪类,透明,块级,行级,行级块标签

CSS伪类专门用来表示标签的一种的特殊的状态，当我们需要为处在这些特殊状态的标签设置样式时，就可以使用伪类 。

伪类的语法：
:link表示普通的链接（没访问过的链接）
:visited表示访问过的链接
浏览器是通过历史记录来判断一个链接是否访问过
使用visited伪类只能设置字体的颜色

:hover伪类表示鼠标移入的状态
:active表示的是被点击的状态

注意：

:visited放在:hover之前
:active 必须被置于 :hover 之后，才是有效的
:hover和:active也可以为其他标签设置
**:focus向拥有键盘输入焦点的标签添加样式。**

定义透明效果的属性是 opacity。
opacity 属性设置标签的不透明级别 值为1。
规定不透明度：从 0.0 （完全透明）到 1.0（完全不透明）。

块级,行级,行级块标签

  • 什么是块级标签？ 

    块级标签：无论内容多少 都会独自占据一行的。
例如<p>、<h1>、<ul>、<ol>、<hr/>等。

  • 什么是行级标签？

    行级标签：只占自身大小的标签，不会占一行。
例如<font>、<b>、<i>、<a>等。

  • 什么是行级块标签？

    例如 <input/> <img>等

注意：

一般情况下使用块级标签包含行级标签，不使用行级标签包含块标签。
a可以包含任何标签，除去a本身；
p标签不可以包含任何的块标签。

块级标签
		     无论内容有多少,都会占据一行
			 默认宽:与父级标签一致
			 默认高:0  或者与内容高度一致
			 
			 还可以通过 width和height设置标签的宽 高
			 
行级标签:
		         只占自身大小,不会占一行
				 即使设置宽 高也是无效的.
				 a
				 
 行级块标签
		可以设置大小,不会占一行
		img <input/>

 一般使用块级标签 包含 行级标签
		a可以包含任何标签，除去a本身；
		p标签不能包含任何块级标签

通过display样式可以修改标签的类型。

可选值：
block ：设置标签为块标签
inline ：设置标签为行级标签
inline-block ：设置标签为行级块标签
none ：隐藏标签（标签将在页面中完全消失） 

<body>
		<!-- 
		 display可以改变标签的类型
		 display: inline; 设置标签为行级标签
		 -->
		<h4 style="display: inline; width: 500px;">4级标题标签</h4>aaaaaaaaaaaaa
		<hr/>
		
		<!-- 
		display: block; 设置标签为块级标签 
		 -->
		<a href="" style="display: block; width: 500px;">百度</a>bbbbbbbbbbbbbbbbb
		
		<hr/>
		<!-- 
		 display:inline-block 设置标签为行级块标签
		 -->
		<a href="" style="display:inline-block; width: 500px;">百度</a>bbbbbbbbbbbbbbbbb
		
		<hr/>
		<!-- display: none; 让标签在网页上消失  不占用网页空间 -->
		<img src="img/bg.jpg" style="display: none;" >CCCCCCCCCCCCCC
	</body>

div和span

div标签
div是块级标签，可以放置任何标签。
div没有任何附加功能，给了什么属性就能变成什么样。
div主要的作用是被用来布局网页。

span标签
span是行级标签
span 没有任何附加功能，给了什么属性就能变成什么样。
span标签被用来选中文档中的文字。

<body>
		<!-- 
		   因为h1 p ....这些标签都有默认的样式,在网页布局时使用比较麻烦
		   
		   div是一个纯净版的块级标签.(没有任何附加样式)
		   可以包含任何标签,
		   主要用来网页布局
		 -->
		 <div style="width: 500px; height: 300px; background-color: #008000;">
			 div是一个纯净版的块级标签
		 </div>
		 aaaaaa
		 
		 <hr/>
		 
		 <!-- 
		     span是行级标签
			 span是一个纯净版的行级标签
		 -->
		 <span style="color: #FF0000;">span是一个纯净版的行级标签</span>
		 <span>span是一个纯净版的行级标签</span>
	</body>
