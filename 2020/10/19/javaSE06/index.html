类

什么是类

  • 类是一个模板,他描述一类对象的属性和行为
  • 表示一个客观世界某类群体的一些基本特征抽象

类=汽车的图纸== .java文件

对象就是具体存在的用图示做出了的汽车

类的结构

  • 成员变量:对象属性的描述(这个类有什么)
  • 方法: 对象的行为(这个类能干什么)
  • 构造方法:用于创建对象
  • 内部类:即在类体中的声明的类
  • 块:代码块

类的定义

类的声明

类的声明格式为:
[访问权限修饰符]  [修饰符] class 类名{	
}
  • java类名的命名规范:类名首字母大写,见名知意,驼峰表示
  • 关键字class用来定义一个类

写出类的共有属性

public class Method {

    /**
     * 类的属性  类的成员   变量
     * 导弹
     */
    String name;//名字
    String colour;//颜色
    int length;//长度
}
  • 变量的格式为：
    [访问权限修饰符] [修饰符] type attr_name [=defaultValue]

类的方法

方法声明格式为:
 [访问权限修饰符] [修饰符]/ void  start(){
	System.out.println("启动");
               [return   返回值;]
     }

对象

为模板在内存中创建的实实在在存在的实例。

创建和使用

  • 使用new关键字创对象.

  • Method testMethod =new Method(); new 关键字+默认构造方法();在内存空间创建一个对象 实际存在的 实例化对象

  • Method testMethod以Method类作为类型 声明一个变量 指向内存空间中的具体的对象使用testMetho变量访问对象中属性,方法

  • 通过类可以创建无数个对象

使用对象 . 成员变量来引用对象的成员变量testMethod.name

使用对象（引用） . 方法来调用对象的方法testMethod.start()

同一类的每个对象有不同的成员变量存储空间,同一类的每个对象共享该类的方法。

创建对象的过程也叫实例化对象. 对象==实例

现实生活中先有对象后有类,而编程时先设计类后创建对象

package java.ff.javaSE.oop.day1;


/**
 * 发现类
 * 类是一个文件,记录信息
 * 声明语法:
 * [访问权限修饰符 public/default][修饰符 final/abstract] class 类名{  }
 */
public class Method {

    /**
     * 类的属性  类的成员   变量
     * 导弹
     */
    String name;
    String colour;
    int length;

    /**
     * 攻击
     * 类的行为  类的成员 方法  被类包含
     * [访问权限修饰符][修饰符] 返回值类型/void 方法名([参数类型 参数名称]){
     * [return 值];
     * }
     */
    public void start(String name, int length) {
        System.out.println("导弹名字:" + name + "攻击范围是:" + length);

    }
}
public class Test {
    public static void main(String[] args) {
        Method   testMethod   = new Method();
         testMethod.name="adsa";
         testMethod.start("东方",100000);
    }
}

成员变量

  • 定义类中,方法外
  • 包括java中的任意一个数据变量(基本数据类型和引用数据类型)
  • 变量可以不被赋值,java虚拟机会给其初始值
  • 成员变量在闯进对象的时候会从类中赋值一份到对象中
  • 成员变量可以被类中方法、构造方法和特定类的语句块访问。
  • 成员变量的作用范围为整个类体

局部变量:

  • 在构造方法,方法或者语句中定义的变量称为局部变量
  • 包括java中的任意一个数据变量(基本数据类型和引用数据类型)
  • 在局部变量使用前必须赋值,
  • 方法结束后,会自动销毁,不占用空间

一个类可以包含以下类型方法：

成员方法

成员方法是定义在类中。这种方法在创建对象的时候创建。
语法格式：

 	修饰符  返回值类型/void  方法名 ( 参数列表) {
  	 方法体语句；
       [return 返回值]
}

构造方法

用来创建对象的方法

  • 是定义在java类中的一个用来创建对象的方法
  • 构造方法可以与类名相同,且没有返回值,且不需要使用void进行修饰
  • 一个类中可以有多个构造方法
  • 使用new + 构造方法 创建一个新的对象那个,在创建对象的时候至少调用一个构造方法.
  • 每个类都有构造方法,java一般会默认提供一个无参构造方法,但是只有Java构造一个后就会失效.

类方法

（静态方法）类方法也声明在类中，但必须声明为static类型

方法的重载

  • 是指一个类中有相同的名字,但是参数不同的多个方法

  • 调用时会根据不同的参数表选择对应的方法

  • 类型,数量,顺序,数据类型,同一种功能处理不同的数据类型

  • 参数不同（可以有三方面的不同）
    数量不同
    类型不同
    顺序不同

    注意:方法重载跟方法的返回值类型,访问权限没有任何关系

package com.nie.day1.Demo02;

public class Construction {

    /**
     * 成员变量
     */
    String name;
    int age;
    double heigth;

    /**
     * 方法的重载
     * 是指一个类中有相同的名字,但是参数不同的多个方法,
     * 调用时会根据不同的参数表选择对应的方法\
     * 类型,数量,顺序,数据类型,同一种功能处理不同的数据类型
     */
    public void eat() {
        System.out.println();
    }

    public void eat(String name) {
        System.out.println("111");
    }


    public void eat(int age) {
        System.out.println("2222");
    }


    public void eat(double age) {
        System.out.println("33333");
    }

    public void eat(float age) {
        System.out.println("44444");
    }

    public void eat(String na, int a) {
        System.out.println("55555");
    }

    public void eat(int a, String na) {
        System.out.println("66666");
    }
}

package com.nie.day1.Demo02;

public class Test {
    public static void main(String[] args) {
        Construction a = new Construction();
        
        a.eat("lllll");
        a.eat(19);
        a.eat(2.22);
        a.eat(2.2f);
        a.eat("sadad",11);
        a.eat(11,"sadad");

    }
}
