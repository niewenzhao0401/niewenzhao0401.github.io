HTML概述

HTML指的是超文本标记语言

  • 超文本：指的是页面内可以包含图片，连接，声音，视频等内容
  • 标记：标签（告诉网页该显示什么内容） 如下所示
<meta name=“keywords” content=“手机,家电">
<meta name="description" content="免费 Web & 编程 教程">
<meta name="author" content=“jim">

标题处添加图标

<link rel="icon" href="ico地址">
  • rel:声明是什么类型 herf：地址

注释后的内容不会显示在网页上

html5的文档声明

如果不声明解析网页时会产生一些不可预期的行为，所以我们应该避免出现。
</!doctype>

HTML基本语法

标签：

  • HTML中的标记指的就是标签，HTML使用标记标签来描述页面

结构

  • <标签名>标签内容</标签名> 闭合标签(有标签内容)
  • <标签名/> 自闭合标签 (无标签内容)

标签属性：标签可以拥有属性。属性进一步说明了该标签的显示或使用

特性。如：

  1. 属性的格式 ：属性名 = “属性值“

  2. 属性的位置： <标签名 属性名 = “属性值“ >xxx</标签名>

  3. 添加多个属性： <标签名 属性名 = “属性值“ 属性名 = “属性值“ >xxx </标签名>

基本常用标签

  • 标题标签

    <h1></h1>…..<h6></h6>

  • 段落标签

    <p></p>

  • 换行标签

    <br/>

  • 列表 

    有序列<ol><li></li></ol>    无序列表<ul><li></li><ul>

  • 超链接

    <a></a>

  • 图像标签

    <img/>

p标签定义段落

  • p 标签中的文字默认会独占一行，并且段与段之间会有一个间距。

br标签呈现换行效果

  • 可插入一个简单的换行符。
  • 标签是自闭合标签。

超链接

  • a标签:HTML 使用超级链接与网络上的另一个文档相连，通俗的说就是通过连接来访问其他网页资源。
<a href="标签练习.html" target="_blank">标签联系</a><!--加载新窗口 -->
<a href="超链接.html" target="_self">超链接</a><!-- 正常打开 -->

P标签定义段落

  • p 标签中的文字默认会独占一行，并且段与段之间会有一个间距。

br标签呈现换行符效果

<br/>

可插入一个简单的换行符。
标签是自闭合标签。
请注意， 标签只是简单地开始新的一行,不会像段落一样有上下间距.

ol标签：有序列表

  • li标签:列表项
    type属性：1,A,a,i,I

ul无序列表

  • li列表选项
    type属性： circle,disc,square

在HTML中预留了一些字符。 这些预留字符是不能在网页中直接使用的。

比如< 和 >,我们不能直接在页面中使用< 和 >号，因为浏览器会将它解析为html签。
例如： 会被解析为b标签，空格,再多的空格都会当一个空格处理。为了可以使用这些预留字符，我们必须在html中使用字符转义。 

小于号<    : &lt      * 版权(C) : &  copy;     大于号>:    >     商标(TM)    &trade;          * 空格  &nbsp;      注册商标(R) 	&reg;

  • a标签:HTML 使用超级链接与网络上的另一个文档相连，通俗的说就是通过连接来访问其他网页资源。
<a  href="http://www.baiduc.om"   target="_blank">百度</a>
<a  href="login.html" target =“_blank">登录</a>
