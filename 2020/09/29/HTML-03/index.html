内联框架

内联框架可以在一个窗口内嵌入一个子窗口,从而在子窗口中引入一个外部的页面** 

<iframe  src=# 初始页面的url  name=# 框架名字   width=#  宽度   height=#  高度  frameborder=# 是否存在边> ... </iframe>

​ target=”_self” 在当前窗口打开新页面
​ target=”_blank” 在新窗口打开新面
​ target=”aa” 在指定名称的窗口打开新页面

​ target=”_parent” 在父级窗口打开新页面
​ target=”_top” 在顶级窗口打开新页面

相关代码如下:

index 

<body>
	index	
	<a href="baidu.html" target="">百度</a>
	<iframe src="child1.html" name="aa" width="300" height="200" frameborder="1"></iframe>
	<!-- <iframe src="child1.html"></iframe> -->
</body>

baidu

<body>
	百度
</body>

login

<body>
	登录
</body>

child1

<body>
	child1
	<a href="login.html" target="_parent">退出_parent</a>
	<a href="login.html" target="_top">退出_top</a>
</body>

child2

<body>
	child1
	<a href="login.html" target="_parent">退出_parent</a>
	<a href="login.html" target="_top">退出_top</a>
</body>

注意:两链接的写法:

<a href="baidu.html" target="">百度</a>
<iframe src="child1.html" name="aa" width="300" height="200" frameborder="1"></iframe>

    <option  >无业</option>   默认选中selected = ” selected ”
    <option selected = ” selected ”>教师</option>

多文本域:

<textarea name="textarea"    cols="40"    rows="6">  文本域中的内容     </ textarea >

按钮基本语法

<input  type=“reset”  value=“ 重填 ”>     重置表单内容
<input type="submit"  value="提交">      提交表单内容到服务器
<input type="button" value="按钮">         普通按钮没有功能，只能被点击触发事件用
  • type=”resert” 重置按钮,让表单还原到原始状态
  • type=”submit’’ 向服务器提交表单,触发action
  • type=”button” 普通按钮,主要用于触发事件.
