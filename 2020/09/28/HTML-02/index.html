HTML -02概论

表格:

  • tableb标签:表格标签
  • tr标签:表格中的行
  • th标签:表格的表头
  • td标签:表格单元格

表格的基本概念

表格的属性

  • border[表格] 属性: border=”number”
  • background[背景图像]
  • bgcolor[背景颜色]
  • width height 属性
  • cellpadding[填充]
  • cellspacing[间距]
  • align[表格对应的位置] left center right
  • valign[表格内文字对应的位置] top middle bottom

表的合并

  • colspan属性 跨列合并单元格 colspan=”数字”
  • rowspan属性 跨行合并单元格 rowspan=”数字”

表格标签构成

from标签:表单

网页表单中有许多可以输入或者选择的组件,用户可以在表单中填写信息,最终提交表单,把客户端数据提交至服务器

表单—文本

注:

  • type —->文本输入框 ; value=“张三”—->输入标签的默认值 ; size=“20”—> 文本区的宽 eadonly=“readonly —> 内容只读 placeholder=“请输入—>提示信息 disabled=“disabled”—>内容置灰不可使用
  • 在html中还专门提供一个标签，专门用来选中表单中的提示文字的lable标签,lable有一个for属性，该属性需要指定一个表单项的id值。
  1. type=“password ” —> 密码框 value=“ 初始值 ”
  2. type=”radio” —>单选按钮
  • 注意：
  • 单选框通过name属性来分组，name相同是一组才可实现单选。
  • 单选和复选不能输入内容，所以必须指定value的值
  1. type=“checkbox” —> 复选框 checked=”checked“ —>默认选中 name=”cb2“ —>复选框名
  2. type=“file” —>文件选择框 name=“img” –>文件选择框名 accept=“.文件后缀名,.文件后缀名… “ —>限制文件类型

**默认选中 checked=”checked“ ** 表单-选择

**默认选中 selected = ” selected ” ** 表单-下拉框
