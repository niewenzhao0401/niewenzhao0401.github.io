HTML -02概论

表格:

  • tableb标签:表格标签
  • tr标签:表格中的行
  • th标签:表格的表头
  • td标签:表格单元格

表格的基本概念

<table>定义表格
    <tr>定义表行
        <th>定义表头</th>
    </tr>
    <tr>
    <td>定义单元格</td>
    </tr>
</table>

表格的属性

  • border[表格] 属性: border=”number”
  • background[背景图像]
  • bgcolor[背景颜色]
  • width height 属性
  • cellpadding[填充]
  • cellspacing[间距]
  • align[表格对应的位置] left center right
  • valign[表格内文字对应的位置] top middle bottom

表的合并

  • colspan属性 跨列合并单元格 colspan=”数字”
  • rowspan属性 跨行合并单元格 rowspan=”数字”

注:

表格标签构成

table:表格区域
  border边框
  width  height
  bgcolor
  background
  cellpadding:内容到边框的距离
  cellspacing:单元格与单元格之间的距离
  align="center":控制表格在本行中的水平位置
tr:定义表格行
  height
  bgcolor
  background
  align="center" 让本行所有单元格内容居中
  valign 设置内容的垂直位置  top  middle bottom
td:表格的单元格
  width
  bgcolor
  background
  align="center"
  valign 设置内容的垂直位置  top  middle bottom
th:表头,也是单元格,内容加粗居中显示
<caption>学生信息表</caption> 表格标题
注意:表格中的内容都必须写在单元格中

from标签:表单

网页表单中有许多可以输入或者选择的组件,用户可以在表单中填写信息,最终提交表单,把客户端数据提交至服务器

<form action=“http://www.sohu.com”  method=“post”>
	……
</form>


action: 由服务器上那个处理程序
method: 提交方法:一般为post(安全) get 方法

表单—文本

<lable for="userid">用户名<lable>
 <input  type=“text ”id=“userid”  name=“username” value=“张三”  size=“20“   readonly=“readonly”placeholder=“请输入” disabled=“disabled”>

注:

  • type —->文本输入框 ; value=“张三”—->输入标签的默认值 ; size=“20”—> 文本区的宽 eadonly=“readonly —> 内容只读 placeholder=“请输入—>提示信息 disabled=“disabled”—>内容置灰不可使用
  • 在html中还专门提供一个标签，专门用来选中表单中的提示文字的lable标签,lable有一个for属性，该属性需要指定一个表单项的id值。
<input type=“password ”  value=“ 123456 ”  size=“22”/>
  1. type=“password ” —> 密码框 value=“ 初始值 ”
  2. type=”radio” —>单选按钮
  • 注意：
  • 单选框通过name属性来分组，name相同是一组才可实现单选。
  • 单选和复选不能输入内容，所以必须指定value的值
  1. type=“checkbox” —> 复选框 checked=”checked“ —>默认选中 name=”cb2“ —>复选框名
  2. type=“file” —>文件选择框 name=“img” –>文件选择框名 accept=“.文件后缀名,.文件后缀名… “ —>限制文件类型

**默认选中 checked=”checked“ ** 表单-选择

**默认选中 selected = ” selected ” ** 表单-下拉框

表单-下拉框

<select name="bmon">
    <option selected = ” selected ” >无业</option>   默认选中selected = ” selected ”
    <option>教师</option>
    <option>医生</option>
    <option>工人</option> 			
</select>

注:下拉框的name,id,事件等需要在select中定义，而value需要在option中定义

多文本域:

<textarea name="textarea"    cols="40"    rows="6">  文本域中的内容     </ textarea >

按钮基本语法

<input  type=“reset”  value=“ 重填 ”>     重置表单内容
<input type="submit"  value="提交">      提交表单内容到服务器
<input type="button" value="按钮">         普通按钮没有功能，只能被点击触发事件用

内联框架可以在一个窗口内嵌入一个子窗口,从而在子窗口中引入一个外部的页面

​ 初始页面的url 框架名字 宽度 高度

<iframe      src=#          name=#        width=#          height=#  frameborder=#> ... </iframe>
